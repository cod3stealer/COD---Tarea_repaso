##3. Genera el jar
Genero el Jar desde IntelliJ. Para hacer esto hay que entrar a `Project Structure > Artifacts > +`.
Despues de eso, `Seleccionar JAR > From modules with dependencies > Main Class`
Y, por último, `Build > Build Artifacts`
